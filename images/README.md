# Artefatos de imagem (não versionados no Git)

Este diretório é usado para artefatos grandes de build/teste local.

## Política

- Arquivos `.img` e variantes comprimidas **não são commitados** no GitHub por limite de tamanho.
- O repositório versiona scripts, documentação e fluxo reproduzível.

## Observação

Para distribuição pública de imagens grandes, use release assets externos, torrents, mirror HTTP ou armazenamento de objetos.
